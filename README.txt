##Установка node.js и vue.js

-----------------------------------------------------
Чтобы установить npm (Node Package Manager), вам нужно сначала установить Node.js, поскольку npm поставляется в комплекте с Node.js. Вот шаги для установки Node.js и npm:

Установка Node.js и npm
Перейдите на официальный сайт Node.js: https://nodejs.org/

Скачайте инсталлятор: Вы увидите две версии для скачивания: LTS (Long-Term Support) и Current. Для большинства пользователей рекомендуется скачать версию LTS, так как она более стабильная.

Запустите инсталлятор: После скачивания запустите скачанный файл и следуйте инструкциям на экране. Инсталлятор установит Node.js и npm.

Проверьте установку: Откройте командную строку или терминал и выполните следующие команды, чтобы проверить, что Node.js и npm были установлены корректно:

Copy code
node -v

###установка vue.js и зависимостей 


введите в терминал: npm install -g @vue/cli

так же 
```
		npm install vue
		npm install vuetify
		npm install axios
```
